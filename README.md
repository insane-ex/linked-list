# Linked List in Rust

A doubly linked list implemented in Rust using raw pointers and manual memory management.  
This is only for educational purposes and to study `unsafe` Rust.

## Features

- Push elements to the front (`push_front`)
- Push elements to the back (`push_back`)
- Remove elements from the front (`pop_front`)
- Remove elements from the back (`pop_back`)
- Check if the list is empty (`is_empty`)
- Get the number of elements (`len`)
- Access the front element (`front`)
- Access the back element (`back`)
- Access the front element mutably (`front_mut`)
- Access the back element mutably (`back_mut`)
- Check if a specific element exists (`contains`)
